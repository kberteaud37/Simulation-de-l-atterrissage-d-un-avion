get_float_input

choix_utilisateur.get_float_input.get_float_input(prompt)[source]

Demande à l’utilisateur de saisir une valeur flottante valide.

Cette fonction continue de demander à l’utilisateur de saisir une valeur jusqu’à ce qu’une valeur flottante valide soit entrée. Elle remplace également les virgules par des points pour faciliter la conversion en float.

Paramètres:

prompt (str) – Message affiché à l’utilisateur pour lui demander de saisir une valeur.

Renvoie:

La valeur flottante saisie par l’utilisateur.

Type renvoyé:

float
